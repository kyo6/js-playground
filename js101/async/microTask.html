Click Button to see Console

The Code:

button.addEventListener('click', () => {
  Promise.resolve().then(() => {
    console.log('MicroTask 1')
  })
  console.log('Listener 1')
})
button.addEventListener('click', () => {
  Promise.resolve().then(() => {
    console.log('MicroTask 2')
  })
  console.log('Listener 2')
})

The Result:

当用户点击按钮是会输出：
Listener 1
MicroTask 1
Listener 2
MicroTask 2
当执行Js Script是会输出：
Listener 1
Listener 2
MicroTask 1
MicroTask 2
